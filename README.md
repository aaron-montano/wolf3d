# wolf3d
A ray-casting game engine with music, collectibles, and interact-able environment.

CONTROLS

move: WASD\
look: Arrow keys\
toggle run/walk : TAB\
interact: E\
drop: R\
change item: 1 and 2\
use warp pipe: T\
You win when you find all 3 "42 silicon Valley" Badges.\
